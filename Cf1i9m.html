문제

음식을 주문 받는 문제입니다 이를 해결하는 다음 프로그램의 실행상태를 확인하세요 
주문 도와드리겠습니다, 손님.
메인요리는 고르셨습니까 : 케이준치킨샐러드
소스는 어떻게 할까요 : 허니머스터드드레싱
음료는 무엇으로 준비할까요 : 콜라
메인요리는 케이준치킨샐러드, 소스는 허니머스터드드레싱, 음료는 콜라를 선택하셨습니다. 
준비하겠습니다, 잠시만 기다려 주십시오.

프로그램 코드

		#include <stdio.h>								
	
1	int main() 						
	{
		char mainDish[32] = "";
		char sauce[32] = "";
2		char drink[32] = "";
	
3		printf( "주문 도와드리겠습니다, 손님.\n" );	
	
4		printf( "메인요리는 고르셨습니까: " );		
5		scanf( "%s", mainDish );			
	
6		printf( "소스는 어떻게 할까요: " );		
7		scanf( "%s", sauce );				
	
8		printf( "음료는 무엇으로 준비할까요: " );	
9		scanf( "%s", drink );				
	
10		printf( "메인요리는 %s, 소스는 %s, 음료는 %s를 선택하셨습니다. \n",
		         mainDish, sauce, drink );				
11		printf( "준비하겠습니다, 잠시만 기다려 주십시오." );
	
12		return 0;
	}
