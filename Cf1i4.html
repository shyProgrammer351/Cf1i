문제

공이 굴러가는 애니메이션을 출력하는 문제입니다 이를 해결하는 다음 프로그램을 해석하세요
O

알고리즘

프로그램 시작 
   첫번째 공 출력 
   1초 후 첫번째 공은 빈칸으로 덮어쓰고 옆 칸에 두번째 공 출력 
   1초 후 두번째 공은 빈칸으로 덮어쓰고 옆 칸에 세번째 공 출력 
   1초 후 세번째 공은 빈칸으로 덮어쓰고 옆 칸에 네번째 공 출력 
   1초 후 네번째 공은 빈칸으로 덮어쓰고 옆 칸에 다섯번째 공 출력 
프로그램 종료

프로그램 코드 

	// 파일명 : ./Chapter01/RollingBall.c

	#include <stdio.h>

	#include <unistd.h>

	 

	// 프로그램 시작 

1	int main() {

	 

		// 첫번째 공 출력 

2		printf( " \033[1;1f O" );

3		fflush( stdout );

	 

		// 1초 후 첫번째 공은 빈칸으로 덮어쓰고 옆 칸에 두번째 공 출력 

4		sleep(1);

5		printf( " \033[1;2f O" );

6		fflush( stdout );

	 

		// 1초 후 두번째 공은 빈칸으로 덮어쓰고 옆 칸에 세번째 공 출력 

7		sleep(1);

8		printf( " \033[1;3f O" );

9		fflush( stdout );

	 

		// 1초 후 세번째 공은 빈칸으로 덮어쓰고 옆 칸에 네번째 공 출력 

10		sleep(1);

11		printf( " \033[1;4f O" );

12		fflush( stdout );

	 

		// 1초 후 네번째 공은 빈칸으로 덮어쓰고 옆 칸에 다섯번째 공 출력 

13		sleep(1);

14		printf( " \033[1;5f O" );

15		fflush( stdout );

	 

		// 프로그램 종료 

16		return 0;

	}
