문제

주사위가 굴러가는 애니메이션을 출력하는 문제입니다 이를 해결하는 다음 프로그램을 해석하세요 
+-------+ +-------+ +-------+ +-------+ +-------+ +-------+ 
|       | | O   O | | O   O | | O     | | O   O | | O     | 
|   O   | |       | |   O   | |   O   | | O   O | |       | 
|       | | O   O | | O   O | |     O | | O   O | |     O | 
+-------+ +-------+ +-------+ +-------+ +-------+ +-------+

알고리즘

프로그램 시작 
   1번 주사위 출력 
   1초 후 빈 화면에 2번 주사위 출력 
   1초 후 빈 화면에 3번 주사위 출력 
   1초 후 빈 화면에 4번 주사위 출력 
   1초 후 빈 화면에 5번 주사위 출력 
   1초 후 빈 화면에 6번 주사위 출력
